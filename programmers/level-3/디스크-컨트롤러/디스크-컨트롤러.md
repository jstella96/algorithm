## 디스크 컨트롤러

## 문제 설명

## 문제 풀이

E(A)+E(B)+E(C) -(S(A)+ S(B)+S(C)) = 이게 작은값
E(A) -S(A)

## 자바 코드

```java
class Solution {

	public int solution(int[][] jobs) {

		int answer = 0;
		int end = 0; // 수행되고난 직후의 시간
		int jobsIdx = 0; // jobs 배열의 인덱스
		int count = 0; // 수행된 요청 갯수

		// 원본 배열 오름차순 정렬 (요청시간 오름차순)
		Arrays.sort(jobs, (o1, o2) -> o1[0] - o2[0]);
		//sort 할때 두가지 경우를 같이 정렬가능한가 -

		// 처리 시간 오름차순으로 정렬되는 우선순위 큐(Heap)
		PriorityQueue<int[]> pq = new PriorityQueue<>((o1, o2) ->  o1[1] - o2[1]);
        // PriorityQueue<int[]> pq = new PriorityQueue<>(
        //     new Comparator<Integer> () {
        //         public int compare(int[] o1,int[] o2) {
        //            return o1[1] - o2[1];
        //         }
        //     }
        // );
		// 요청이 모두 수행될 때까지 반복
		while (count < jobs.length) {

			// 하나의 작업이 완료되는 시점(end)까지 들어온 모든 요청을 큐에 넣음
			while (jobsIdx < jobs.length && jobs[jobsIdx][0] <= end) {
                pq.add(jobs[jobsIdx]);
                jobsIdx++;
			}

			// 큐가 비어있다면 작업 완료(end) 이후에 다시 요청이 들어온다는 의미
            // 작업이 끝났는데 대기 요청 없으면, 가장 앞에있는 배열을 가져와요. -> 수행들어감.
			// (end를 요청의 가장 처음으로 맞춰줌)
			if (pq.isEmpty()) {
				end = jobs[jobsIdx][0]; //요청시간으로 셋팅
			// 작업이 끝나기 전(end 이전) 들어온 요청 중 가장 수행시간이 짧은 요청부터 수행
			} else {
				int[] temp = pq.poll(); // 가장 빠른거
				answer += temp[1] + end - temp[0];// (소요시간 + 현재까지 시간 - 요청한시간)
				end += temp[1]; // 시간지나고
				count++;// 한개처리
			}
		}

		return (int) Math.floor(answer / jobs.length);
	}
}
```
